   ```python
   import mlx.core as mx, time
   num_layers, seq_len, num_heads, head_dim = 32, 8192, 8, 128
   kv_cache = [(mx.random.normal([1, seq_len, num_heads, head_dim]),
   ```

4. Computes per-layer latency: `per_layer_ms = min(times) / num_layers`

5. Assert: `per_layer_ms < 10.0`
