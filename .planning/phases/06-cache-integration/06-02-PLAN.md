The parent `save_block()` does the following in order. The override must replicate all steps except the array-building loop, substituting compressed bytes instead:

```
1. Duplicate-check: self._index.contains(block_hash) → return True (hit)
2. Duplicate-check hot cache:
       with self._hot_cache_lock:
           if block_hash in self._hot_cache: return True
3. Queue-capacity guard (SSD path only):
       if not self._hot_cache_enabled and self._write_queue.full(): return False
4. file_path = self._get_file_path(block_hash)
5. (SSD path) self._enforce_size_limit_for_new_block()
6. Build arrays dict and metadata dict (step the override replaces with blob bytes)
7. Materialize: mx.eval(*arrays.values())
8. Extract raw bytes:
       tensors_raw = {}
       for name, arr in arrays.items():
           tensors_raw[name] = _extract_tensor_bytes(arr)
       # _extract_tensor_bytes returns (raw_bytes: bytes, dtype_str: str, shape: List[int])
       # tensors_raw maps tensor_name -> (bytes, dtype_str, shape)
9. Build PagedSSDBlockMetadata (dataclass with fields: block_hash, file_path,
       file_size, token_count, created_at, last_access, num_layers, model_name,
       layer_cache_types, layer_meta_states)
10. Build cache_entry dict:
       cache_entry = {
           'tensors_raw': tensors_raw,        # Dict[str, Tuple[bytes, str, List[int]]]
           'file_metadata': metadata,         # Dict[str, str]
           'num_layers': len(cache_data),     # int
           'layer_cache_types': layer_cache_types,  # Optional[List[str]]
           'block_metadata': block_metadata,  # PagedSSDBlockMetadata
       }
11a. Hot-cache path (self._hot_cache_enabled == True):
       self._hot_cache_put(block_hash, cache_entry)
       self._stats["saves"] += 1
       return True
11b. SSD path (self._hot_cache_enabled == False):
       self._index.add(block_metadata)
       with self._hot_cache_lock:
           self._hot_cache[block_hash] = cache_entry
       with self._pending_write_hashes_lock:
           self._pending_write_hashes.add(block_hash)
       try:
           self._write_queue.put_nowait(
               (block_hash, tensors_raw, metadata, file_path)
           )
       except queue.Full:
           self._index.remove(block_hash)
           self._hot_cache_remove(block_hash)
           return False
       self._stats["saves"] += 1
       return True
```

For the compressed override, intercept at the point where `compressed` flag is detected:
- In the hot-cache path (step 1): after getting `entry`, check `entry['file_metadata'].get('compressed') == 'true'`. If so, reconstruct `PipelineBlob` from the stored blob bytes and call `self._pipeline.decompress(blob)`. Return the result. If decompress raises, log error and return None.
- In the disk path (step 4-6): after `mx.load()`, check `file_metadata.get('compressed') == 'true'`. If so, get `arrays['compressed_blob']`, convert to bytes via `bytes(memoryview(np.array(arrays['compressed_blob'])))`, reconstruct `PipelineBlob(compressed=..., logical_seq_len=int(file_metadata['logical_seq_len']), compaction_ratio=1.0)`, call `self._pipeline.decompress(blob)`. Wrap in try/except; on error log and return None.
- When NOT compressed (`file_metadata.get('compressed')` is absent or not `'true'`): call `super().load_block(block_hash)` to handle normally.

```python
def _hot_cache_get(self, block_hash: bytes) -> Optional[Dict]:
    with self._hot_cache_lock:
        if block_hash in self._hot_cache:
            self._hot_cache.move_to_end(block_hash)
            return self._hot_cache[block_hash]
        return None
```

**load_block() override:**

Check hot cache first via `self._hot_cache_get(block_hash)` (thread-safe, updates LRU order):
```python
entry = self._hot_cache_get(block_hash)
if entry is not None:
    if entry['file_metadata'].get('compressed') == 'true':
        # Reconstruct PipelineBlob from stored raw bytes
        raw_bytes, dtype_str, shape = entry['tensors_raw']['compressed_blob']
        try:
            from omlx.compression.pipeline import PipelineBlob
            blob = PipelineBlob(
                compressed=raw_bytes,
                logical_seq_len=int(entry['file_metadata']['logical_seq_len']),
                compaction_ratio=1.0,
            )
            result = self._pipeline.decompress(blob)
            self._index.touch(block_hash)
            self._stats["loads"] += 1
            self._stats["hits"] += 1
            self._stats["hot_cache_hits"] += 1
            return result
        except Exception as exc:
            logger.error(f"Decompression failed (hot cache) for {block_hash.hex()[:16]}: {exc}")
            return None
    else:
        # Not compressed: use parent's hot-cache reconstruction
        return super().load_block(block_hash)
```

For the disk path, override continues after the hot-cache check returns None:
```python
# Check index
metadata = self._index.get(block_hash)
if metadata is None:
    self._stats["misses"] += 1
    return None
file_path = metadata.file_path
if not file_path.exists():
    logger.warning(f"SSD cache file missing: {file_path}")
    self._index.remove(block_hash)
    self._stats["misses"] += 1
    return None

arrays, file_metadata = mx.load(str(file_path), return_metadata=True)

if file_metadata.get('compressed') == 'true':
    try:
        import numpy as np
        from omlx.compression.pipeline import PipelineBlob
        blob_arr = arrays['compressed_blob']
        mx.eval(blob_arr)
        raw_bytes = bytes(memoryview(np.array(blob_arr)))
        blob = PipelineBlob(
            compressed=raw_bytes,
            logical_seq_len=int(file_metadata['logical_seq_len']),
            compaction_ratio=1.0,
        )
        result = self._pipeline.decompress(blob)
        self._index.touch(block_hash)
        self._stats["loads"] += 1
        return result
    except Exception as exc:
        logger.error(f"Decompression failed (disk) for {block_hash.hex()[:16]}: {exc}")
        return None
else:
    # Not compressed — fall through to parent's full disk reconstruction
    return super().load_block(block_hash)
```

Note: `super().load_block()` will re-check the hot cache and re-read the disk file. For the non-compressed fallback this is acceptable (rare path). The compressed path is fully handled inline above without calling super().
