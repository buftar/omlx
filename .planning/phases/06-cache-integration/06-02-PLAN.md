**CompressedPagedSSDCacheManager** subclasses PagedSSDCacheManager:

Constructor: accepts all PagedSSDCacheManager args plus `compression_config: CompressionConfig`. Store as `self._compression_config`. Set `self.__pipeline = None` for lazy init.

Pipeline property (lazy init, no MLX import at class-load time):
```python
@property
def _pipeline(self):
    if self.__pipeline is None:
        from omlx.compression.pipeline import KVCachePipeline
        cfg = self._compression_config
        kwargs = dict(
            bundle_path=cfg.bundle_path,
            am_ratio=cfg.am_ratio,
            n_sink_tokens=cfg.n_sink_tokens,
            sliding_window=cfg.sliding_window,
        )
        if cfg.n_components is not None:
            kwargs["n_components"] = cfg.n_components
        self.__pipeline = KVCachePipeline(**kwargs)
    return self.__pipeline
```

**save_block() override:**
- If `not self._compression_config.enabled`: return `super().save_block(block_hash, cache_data, token_count, model_name, layer_cache_types, layer_meta_states)`
- If enabled:
  1. Call `blob = self._pipeline.compress(cache_data)` on the inference thread
  2. Import numpy as np. Encode: `import mlx.core as mx; blob_arr = mx.array(np.frombuffer(blob.compressed, dtype=np.uint8))`
  3. Materialize: `mx.eval(blob_arr)` — this is MLX graph materialization (not Python built-in), safe on inference thread
  4. Extract raw bytes using the inherited `_extract_tensor_bytes(blob_arr)` helper
  5. Build `tensors_raw = {"compressed_blob": raw_bytes_result}`
  6. Build metadata dict with all standard fields plus `"compressed": "true"` and `"logical_seq_len": str(blob.logical_seq_len)`
  7. Replicate parent's hot-cache entry construction and write queue enqueue logic (inspect parent save_block carefully for the full flow: index check, queue capacity, hot cache entry, _write_queue.put_nowait)
  8. Return True

**load_block() override:**
- Check hot cache first (acquire _hot_cache_lock): if block_hash in _hot_cache:
  - Get entry, check `entry['file_metadata'].get('compressed') == 'true'`
  - If compressed: reconstruct PipelineBlob from the stored blob bytes, call decompress, handle errors
  - If not compressed: fall through to super().load_block()
- On disk path: call `mx.load(file_path, return_metadata=True)` to get arrays and metadata
  - If `file_metadata.get("compressed") == "true"`: get the "compressed_blob" array, convert to bytes, reconstruct PipelineBlob(compressed=..., logical_seq_len=int(file_metadata["logical_seq_len"]), compaction_ratio=1.0), call `self._pipeline.decompress(blob)`
  - Return the decompress result (list of layers)
- On any exception during decompress: `logger.error(...)`, return None (cache miss per CONTEXT.md locked decision)

Note: Inspect the actual load_block() implementation in paged_ssd_cache.py to understand the exact hot-cache lookup pattern and disk-read path. The override only diverges at the point where compressed metadata is detected.
