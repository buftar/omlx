**`measure_decompression_latency(pipeline, blob, n_layers, n_warmup=1)`:**
- Import time and mlx.core inside function
- Warmup loop (n_warmup iterations): call `pipeline.decompress(blob)`, unpack `(layers, _)`, force MLX materialization with `mx.eval(*[t for pair in layers for t in pair])`
- Timed run: `t0 = time.perf_counter()`, decompress, mx.eval all tensors, `elapsed_ms = (time.perf_counter() - t0) * 1000`
- Return `elapsed_ms / n_layers`

**`run_litm(model, tokenizer, pipeline, n_problems=100, seed=42)`:**
- `rng = np.random.default_rng(seed)` — use for all random generation
- Generate 100 LITM problems: each has 20 short passages, code buried at position 10
  - Filler passage: `f"Document {i}: This passage contains general information about topic {rng.integers(100, 999)}."` (50-60 tokens)
  - Target passage at index 10: `f"Document 10: The secret code is {code}."` where `code = int(rng.integers(1000, 9999))`
  - Question: `"Based on the documents, what is the secret code?"`
- For each problem, run two paths:
  - Vanilla: build prompt_cache from the context passages via model prefill, then generate up to 32 tokens on the question, check if `str(code)` appears in decoded output
  - Compressed: build same prompt_cache, extract kv_cache, `pipeline.compress(kv_cache)` then `pipeline.decompress(blob)`, inject decompressed layers into fresh cache, generate on question
- Return `{"litm_vanilla": vanilla_correct/n_problems, "litm_compressed": compressed_correct/n_problems}`

**`run_gsm8k(model, tokenizer, pipeline, n_samples=200, seed=42)`:**
- `from datasets import load_dataset` inside function
- Load `load_dataset("openai/gsm8k", "main", split="test")` — dataset is cached by HF after first download
- Sample with `np.random.default_rng(seed).choice(len(ds), min(n_samples, len(ds)), replace=False)`
- Prompt template: `"Solve step by step. End with: #### <number>\n\nProblem: {question}\nSolution:"`
- Run two paths (vanilla and compressed) for each sample
- Extract answer: `import re; m = re.search(r'####\s*([\d,.-]+)', text); answer = m.group(1).replace(',','').strip() if m else None`
- Gold answer: `item["answer"]` contains the `####` marker — extract with same regex
- Return `{"gsm8k_vanilla": vanilla_correct/n_samples*100, "gsm8k_compressed": compressed_correct/n_samples*100}`

**`run_mmlu(model, tokenizer, pipeline, n_samples=200, seed=42)`:**
- Subjects: `["elementary_mathematics", "high_school_physics", "moral_scenarios", "professional_medicine", "computer_security"]`
- Load each: `load_dataset("cais/mmlu", subject, split="test")`, sample `n_samples // 5` per subject
- Prompt: `"Question: {question}\nA) {choices[0]}\nB) {choices[1]}\nC) {choices[2]}\nD) {choices[3]}\nAnswer:"`
- Generate 1 token; compare first non-whitespace character to `["A","B","C","D"][item["answer"]]`
- Run vanilla and compressed paths for each sample
- Return `{"mmlu_vanilla": correct/total*100, "mmlu_compressed": correct/total*100}`

Anti-patterns from research to avoid:
- NEVER share cache objects between vanilla and compressed paths — always call `make_prompt_cache(model)` fresh
- ALWAYS force materialization with `mx.eval()` before stopping latency timer
- For cosine sim: compare compacted vs decompressed (not original vs compacted)

```python
QWEN_MODEL = "Qwen/Qwen2.5-7B-Instruct"

class TestSlowQwen:
    @pytest.mark.slow
    def test_am_cosine_sim(self):
        runner = BenchmarkRunner(QWEN_MODEL, seed=42)
        report = runner.run_benchmark(tasks=["cosine_sim"])
        sim = report["technical_metrics"]["am_cosine_similarity"]
        assert sim > 0.998, f"AM cosine sim {sim:.4f} below 0.998 threshold"
        assert report["thresholds"]["am_cosine_sim_pass"] is True

    @pytest.mark.slow
    def test_task_accuracy(self):
        runner = BenchmarkRunner(QWEN_MODEL, seed=42, n_samples=50)
        report = runner.run_benchmark(tasks=["gsm8k", "mmlu", "litm"])
        gsm8k_delta = abs(report["quality_metrics"]["gsm8k_vanilla"] - report["quality_metrics"]["gsm8k_compressed"])
        mmlu_delta = abs(report["quality_metrics"]["mmlu_vanilla"] - report["quality_metrics"]["mmlu_compressed"])
        litm_delta = abs(report["quality_metrics"]["litm_vanilla"] - report["quality_metrics"]["litm_compressed"])
        assert gsm8k_delta <= 1.0, f"GSM8K delta {gsm8k_delta:.2f} > 1.0pp"
        assert mmlu_delta <= 1.0, f"MMLU delta {mmlu_delta:.2f} > 1.0pp"
        assert litm_delta <= 0.05, f"LITM delta {litm_delta:.3f} > 0.05"

    @pytest.mark.slow
    def test_full_pipeline_runs(self):
        runner = BenchmarkRunner(QWEN_MODEL, seed=42, n_samples=10)
        report = runner.run_benchmark(tasks=["cosine_sim", "latency"])
        assert report["schema_version"] == "1.0"
        assert report["model"] == QWEN_MODEL
        assert "decompression_latency_ms_per_layer" in report["technical_metrics"]
        assert report["overall_pass"] is not None
```
