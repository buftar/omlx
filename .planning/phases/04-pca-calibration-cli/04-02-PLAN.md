```python
import numpy as np
import mlx.core as mx
from scipy.linalg import orthogonal_procrustes
from omlx.compression.linalg_utils import svd_f32

# MLX tensor graph flush -- forces MLX lazy compute graph to materialize tensors.
# Named _mx_materialize to match convention in am.py (lines 29-30) and kvtc.py.
# This is NOT Python built-in evaluation -- it is mlx.core.eval(), the MLX graph flush.
_mx_materialize = mx.eval  # noqa: S307

N_SVD_TOKENS = 4000   # max tokens fed to SVD -- prevents OOM on unified memory
CALIB_SEED = 42       # fixed seed for deterministic subsampling
```

Implementation notes for each function:

**strip_rope_from_keys** (Pattern 3 from RESEARCH.md):
- Cast input to float32 before trig: ensure np.float32 arrays throughout
- Compute theta_freqs: 1.0 / (rope_theta ** (2 * i / head_dim)) for i in arange(head_dim//2)
- angles = outer(positions, theta_freqs).astype(float32)
- cos_a / sin_a shape: [1, 1, T, head_dim//2]
- Non-traditional (traditional=False): first-half / second-half split
  - k1 = keys[..., :half], k2 = keys[..., half:]
  - return np.concatenate([k1*cos_a + k2*sin_a, -k1*sin_a + k2*cos_a], axis=-1)
- Traditional (traditional=True): consecutive pairs
  - k_even = keys[..., 0::2], k_odd = keys[..., 1::2]
  - out[..., 0::2] = k_even*cos_a + k_odd*sin_a
  - out[..., 1::2] = -k_even*sin_a + k_odd*cos_a

**compute_pca_basis** (Pattern 4 from RESEARCH.md):
- rng = np.random.default_rng(seed=seed)
- Subsample: if len(vectors) > N_SVD_TOKENS: idx = rng.choice(len, N_SVD_TOKENS, replace=False)
- mu = sample.mean(axis=0)
- centered = (sample - mu).astype(np.float32)
- data_mlx = mx.array(centered)
- U, S, Vt = svd_f32(data_mlx)   # mandatory -- bare mx.linalg.svd is forbidden
- _mx_materialize(Vt, S)
- Vt_np = np.array(Vt)   # [min(N, head_dim), head_dim]
- V = Vt_np[:n_components].T  # [head_dim, n_components]
- return V.astype(float32), mu.astype(float32), S_np[:n_components].astype(float32)

**assign_layer_groups** (Pattern 7 from RESEARCH.md):
- Use np.array_split(np.arange(n_layers), n_groups)
- Return [g.tolist() for g in result]

**align_bases_to_reference** (Pattern 5 from RESEARCH.md):
- ref = bases[reference_idx].astype(np.float64)
- For i, B in enumerate(bases): if i == reference_idx: keep; else R, _ = orthogonal_procrustes(B.astype(float64), ref); aligned[i] = (B.astype(float64) @ R).astype(float32)

**save_calibration_bundle**:
- np.savez(str(output_path), **bundle_arrays)
- print(f"Calibration bundle saved: {output_path}")

**load_calibration_bundle**:
- data = np.load(str(path))
- group_sizes = data["group_sizes"]
- Build layer_to_group: iterate over group_sizes, assign layer indices
- For each layer l from 0 to sum(group_sizes)-1: g = layer_to_group[l]
  - append dict with K_basis=data["K_V"][g], K_mean=data["K_mu"][g], K_sv=data["K_sv"][g],
    V_basis=data["V_V"][g], V_mean=data["V_mu"][g], V_sv=data["V_sv"][g],
    k_bit_alloc=data["K_bit_alloc"][g], v_bit_alloc=data["V_bit_alloc"][g]
- return pca_bundle, data["head_entropy"].tolist()
