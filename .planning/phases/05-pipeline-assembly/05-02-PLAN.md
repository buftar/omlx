Implement the four methods in `omlx/compression/pipeline.py`, replacing the `NotImplementedError` stubs from Plan 01:

**`_strip_rope(self, compacted_layers)`:**
- When `self._rope_params is None` (bundle_path=None path): return `compacted_layers` unchanged (skip stripping)
- When rope params are available: for each `(keys, values)` in compacted_layers:
  1. `_mx_materialize(keys)` — force MLX graph execution before numpy bridge (see Pitfall 5 in RESEARCH.md)
  2. `keys_np = np.array(keys.astype(mx.float32))` — convert to float32 numpy
  3. `keys_stripped_np = strip_rope_from_keys(keys_np, rope_theta, rope_traditional, offset=0)` — use offset=0 (correct per RESEARCH.md Pitfall 1)
  4. `stripped_keys = mx.array(keys_stripped_np.astype(np.float16))` — convert back to float16 mx.array (see Pitfall 2 in RESEARCH.md — must be mx.array, not numpy, when passed to kvtc)
  5. Append `(stripped_keys, values)` to result
- Store rope params as `self._rope_params = (rope_theta, rope_traditional)` in `__init__` when `bundle_path is not None`; leave as `None` when `bundle_path is None`

**`compact(self, kv_cache, ratio=None, queries=None)`:**
- `effective_ratio = ratio if ratio is not None else self._am_ratio`
- Return `self._am.compact(kv_cache, ratio=effective_ratio, queries=queries)`

**`compress(self, kv_cache, queries=None)`:**
- `compacted = self._am.compact(kv_cache, ratio=self._am_ratio, queries=queries)`
- `original_seq_len = compacted.logical_seq_len`
- `compacted_seq_len = compacted.layers[0][0].shape[2]` — physical token count after AM
- `actual_ratio = original_seq_len / compacted_seq_len if compacted_seq_len > 0 else 1.0` — compute from actual result, not constructor param (see Pitfall 3 in RESEARCH.md)
- `stripped_layers = self._strip_rope(compacted.layers)`
- `compressed_bytes = self._kvtc.compress(stripped_layers)` — pass list of (mx.array, mx.array), NOT the AMCompactedCache object (see Pitfall 4 in RESEARCH.md — pass `.layers`, not the dataclass)
- Return `PipelineBlob(compressed=compressed_bytes, logical_seq_len=original_seq_len, compaction_ratio=actual_ratio)`

**`decompress(self, blob: PipelineBlob)`:**
- `layers = self._kvtc.decompress(blob.compressed)` — returns list[tuple[mx.array, mx.array]] at compacted token count
- Return `(layers, blob.logical_seq_len)` — return compacted layers + original T for RoPE offset use
