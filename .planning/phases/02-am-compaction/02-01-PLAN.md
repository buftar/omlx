TestUniformFallback (AM-01):
  - test_queries_none_uses_uniform: compact(kv_cache, queries=None) runs without error and returns AMCompactedCache
  - test_uniform_coverage: selected indices are evenly distributed (no clustering)

TestNNLSBetaFitting (AM-02):
  - test_betas_nonneg_before_clip: raw NNLS output is all >= 0 (scipy NNLS guarantee)
  - test_beta_shape_matches_budget: betas shape == (budget,) for each head

TestOLSValueFitting (AM-03):
  - test_v_compact_shape: V_compact shape is [1, 1, budget, head_dim]
  - test_no_bare_pinv_in_am: am.py does not contain 'mx.linalg.pinv' (lint check)

TestCompactedCacheShape (AM-04):
  - test_logical_seq_len_preserved: result.logical_seq_len == input seq_len T
  - test_physical_tokens_reduced: all result.layers[i][0].shape[2] == budget (< T)
  - test_layer_count_preserved: len(result.layers) == len(input kv_cache)

TestHeadBudgets (AM-05):
  - test_uniform_budgets_correct: with head_entropy=None, budget == max(n_sinks, floor(T/ratio)) per head
  - test_entropy_budgets_proportional: with head_entropy=[1.68, 0.34, 2.47, 1.12], higher-entropy heads get more tokens
  - test_entropy_budgets_min_sinks: no budget is below n_sink_tokens

TestBudgetReuse (AM-06):
  - test_budgets_computed_at_init: AMCompactor.__init__ with head_entropy computes budget schedule; compact() does not recompute

TestGenerateReferenceQueries (AM-07):
  - test_sample_method_shape: generate_reference_queries(keys, n_queries=64, method="sample").shape == [1, n_heads, 64, head_dim]
  - test_random_method_shape: generate_reference_queries(keys, n_queries=32, method="random").shape == [1, n_heads, 32, head_dim]

TestBetaBoxConstraint (AM-08):
  - test_betas_clipped_to_minus3_plus3: all betas after compact() are in [-3.0, 3.0]

TestCompactIntegration (AM-02+AM-03):
