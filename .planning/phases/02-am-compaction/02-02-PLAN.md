  pinv_f32(a: mx.array) -> mx.array
    Moore-Penrose pseudoinverse. Handles float16 cast + CPU stream internally.

  nnls_solve(A: mx.array, b: mx.array) -> tuple[mx.array, float]
    Non-negative least squares via scipy bridge.
    A: 2D mx.array [m, n], b: 1D mx.array [m]
    Returns: (solution: mx.array [n] float32, residual: float)
    Raises ValueError if b is not 1D or shapes mismatch.
    Internally calls numpy conversion — caller must force MLX graph materialization
    (via mx.eval) on attn_full, attn_selected, output_full BEFORE calling nnls_solve.

MLX KV cache shapes (from spike_am.py):
  keys, values:                   [1, n_heads, seq_len, head_dim]
  keys_h, values_h (single head): [1, 1, seq_len, head_dim]
  queries_h (single head):        [1, 1, n_queries, head_dim]

Attention computation pattern (from spike - do not deviate):
  scale = head_dim ** -0.5
  scores = (queries_h @ keys_h.transpose(0, 1, 3, 2)) * scale  # [1,1,n_q,seq_len]
  attn = mx.softmax(scores, axis=-1)

NNLS design matrix (from research - use exactly this):
  A_s      = attn_sel[0, 0]              # [n_queries, budget]
  target_b = attn_full[0, 0].sum(axis=1) # [n_queries] row sums of softmax
  betas, residual = nnls_solve(A_s, target_b)
  betas = mx.clip(betas, -3.0, 3.0)      # AM-08 box constraint

OLS value fitting (from research - use exactly this):
  out_target = output_full[0, 0]         # [n_queries, head_dim]
  A_pinv     = pinv_f32(A_s)             # [budget, n_queries]
  V_compact  = A_pinv @ out_target       # [budget, head_dim]
  V_compact  = V_compact[None, None]     # [1, 1, budget, head_dim]
  mx.eval(V_compact)

  <action>
Replace the NotImplementedError stub in compact() and implement _compact_head as a private method.

_compact_head implementation order:
1. Compute budget = max(self.n_sink_tokens, min(budget, seq_len)) as a guard.
2. If queries_h is None: call _uniform_select, return (keys_h[:,:,selected], values_h[:,:,selected]) — no fitting.
3. If queries_h provided: call _highest_attn_select to get selected indices.
4. Extract k_sel = keys_h[:, :, selected].
5. Compute full attention weights (scale = head_dim**-0.5, softmax of q@k.T).
6. Compute selected attention weights (softmax of q@k_sel.T).
7. Compute full attention output = attn_full @ values_h.astype(mx.float32).
8. Force graph materialization on output_full, attn_full, attn_sel via mx.eval before nnls_solve.
9. NNLS beta-fitting: A_s = attn_sel[0,0], target_b = attn_full[0,0].sum(axis=1), call nnls_solve, clip to [-3, 3].
10. OLS value fitting: pinv_f32(A_s) @ output_full[0,0], reshape to [1, 1, budget, head_dim].
11. Return (k_sel, V_compact).

_highest_attn_select helper:
1. Compute attention scores and softmax.
2. Sum across query axis to get per-token importance: attn[0,0].sum(axis=0).
3. Always include first n_sink_tokens indices.
4. From non-sink positions, pick top (budget - n_sinks) by mx.argsort (ascending, take last n_select).
5. Combine sink indices and non-sink topk, mx.sort the result.
6. Force materialization via mx.eval, return selected.

_uniform_select helper:
1. Compute sink_idx = list(range(min(n_sink_tokens, seq_len))).
2. remaining = budget - len(sink_idx); if <= 0, return mx.array(sorted(sink_idx)).
3. non_sink = np.linspace(n_sinks, seq_len-1, remaining, dtype=int).
4. Merge sink_idx and non_sink via set union, sort, wrap as mx.array.
5. Force materialization via mx.eval, return.

compact() implementation:
1. Guard: empty kv_cache raises ValueError.
2. Extract seq_len and n_heads from first layer keys.shape.
3. Compute budget_per_head = max(self.n_sink_tokens, int(seq_len / ratio)).
4. Outer loop over layers, inner loop over heads (h in range(n_heads)).
5. Slice k_h = keys[:, h:h+1], v_h = values[:, h:h+1].
6. Slice q_h = queries[:, h:h+1] if queries is not None else None.
7. Call _compact_head(k_h, v_h, q_h, budget_per_head), collect results.
8. Concatenate heads on axis=1 back to [1, n_heads, budget, head_dim].
9. Return AMCompactedCache(layers=compacted_layers, logical_seq_len=seq_len, diagnostics=None).

Note: compact() in this plan uses uniform budgets. Plan 03 adds _compute_head_budgets and upgrades to non-uniform when head_entropy is not None.
